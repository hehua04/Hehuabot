# Hehuabot

A bot project for automated Postal Wiki maintenance tasks.

## Features

- **add_category.py** - Add categories to pages
- **categorize_files.py** - Categorize files by type
- **categorize_files_city.py** - Categorize city-related files
- **categorize_files_province.py** - Categorize province-related files
- **fix_municipality.py** - Fix municipality-related formatting
- **remove_main_template.py** - Remove main template from pages
- **zhpw_family.py** - Handle zhpw family templates

## Usage

Run each script individually:

```bash
python add_category.py
python categorize_files.py
python categorize_files_city.py
python categorize_files_province.py
python fix_municipality.py
python remove_main_template.py
python zhpw_family.py
```
